Apa itu HTML5?

HTML5 adalah versi terbaru dari HTML (HyperText Markup Language) yang digunakan untuk membuat dan menampilkan halaman web.

Keunggulan HTML5

  • Struktur kode lebih semantik
  • Mendukung video dan audio tanpa plugin
  • Responsif untuk berbagai perangkat
